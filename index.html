Welcome to Simple Swarm Simulator (SSS)

Written in python with pygame

Keys and Features

Array Up - Faster
Arrow Down - Slower
Arrow Right - Next Mode
Arrow Left - Previous mode
Key P - Show/Hide Path
ESC - quit

Do Not Press the B Key while running, Very Dangerous.

Installation and run

python 3.4
pygame 2.8
run with
open terminal to swarmsim/componets/
python ./main.py
